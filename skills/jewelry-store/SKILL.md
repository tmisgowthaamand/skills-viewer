## Business Overview

Jewelry retail and custom manufacturing.

## Industry at a Glance

| Item | Details |
|------|---------|
| **Type** | Retail |
| **Revenue Model** | Gold/silver/diamond sales, making charges |
| **Key Costs** | Inventory (gold), rent, staff, security |
| **Typical Team Size** | 3–15 |
| **Licenses Needed** | BIS hallmark, GST, Shop & Est. |
| **Primary Tools** | Jewelry ERP (Jewel Guru), POS, Tally |

## Quick Start Checklist

- [ ] Register business (GST + local licences)
- [ ] Set up Jewelry ERP (Jewel Guru) as primary tool
- [ ] Create service/product menu with pricing
- [ ] Open Google My Business
- [ ] Set up WhatsApp Business
- [ ] Define 3 core SOPs
