## Business Overview

A software product or services company building SaaS applications, enterprise software, mobile apps, or custom development solutions for businesses and consumers.

## Industry at a Glance

| Item | Details |
|------|---------|
| **Type** | Technology / SaaS / IT Services |
| **Revenue Model** | SaaS subscriptions (MRR/ARR), project-based, maintenance contracts |
| **Key Costs** | Engineering salaries, cloud infrastructure, sales & marketing |
| **Typical Team Size** | 5–200 people |
| **Licenses Needed** | Company registration, GST, ISO 27001 (for enterprise), CMMI |
| **Primary Tools** | GitHub, Jira, Confluence, AWS/GCP/Azure, Figma, Slack, HubSpot |

## Quick Start Checklist

- [ ] Register company (Pvt Ltd)
- [ ] Get GST registration
- [ ] Set up GitHub organisation and CI/CD pipelines
- [ ] Define product roadmap and MVP scope
- [ ] Set up CRM (HubSpot) for sales pipeline
- [ ] Apply for Startup India recognition
- [ ] Set up cloud infrastructure (AWS/GCP)
