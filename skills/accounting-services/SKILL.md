## Quick Start Checklist

- [ ] Register CA firm with ICAI (or partner with a CA)
- [ ] Get GST registration on GST portal
- [ ] Set up TallyPrime/Zoho Books for client billing
- [ ] Create standard chart of accounts templates
- [ ] Build client onboarding packet (NDA + engagement letter)
- [ ] Define 5-tier service menu with pricing
- [ ] Set up Google My Business listing
- [ ] Create social media pages (LinkedIn, Instagram, YouTube, X)
- [ ] Implement CRM (Zoho CRM / HubSpot free)
- [ ] Launch with first 20–50 clients via referrals
