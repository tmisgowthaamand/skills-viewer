## Business Overview

International trade in goods.

## Industry at a Glance

| Item | Details |
|------|---------|
| **Type** | Trade |
| **Revenue Model** | Trading margin, commission |
| **Key Costs** | Working capital, freight, customs duties |
| **Typical Team Size** | 3–20 |
| **Licenses Needed** | DGFT IEC code, GST |
| **Primary Tools** | DGFT portal, ICEGATE, SAP, IndiaMART |

## Quick Start Checklist

- [ ] Register business (GST + local licences)
- [ ] Set up DGFT portal as primary tool
- [ ] Create service/product menu with pricing
- [ ] Open Google My Business
- [ ] Set up WhatsApp Business
- [ ] Define 3 core SOPs
